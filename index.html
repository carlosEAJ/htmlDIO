Como o html é formado?

O html consiste em 2 elementos que são: head(cabeça) e o body(corpo) e tudo que estiver dentro do body (assim como esse texto) vai aparecer para o cliente.

Vale ressaltar que, as tags citadas acima sempre vão estar dentro da tag "pai" html.

Tópicos

  1. Tags
  2. atributos
  3. listas
  4. link

Tags

html: define o que é página e tudo que estiver dentro dela vai ser uma.

i: transforma o texto em itálico

strong: transforma o texto em negrito

img: adicionam uma imagem ao site e necessita da tag de atributo "src", o src necessita adiciona a imagem atraves do diretorio, seja ele externo ou interno.

h1, h2...: header que é o titulo da página e conforme o número aumenta o nível do título diminui

P: define o corpo do texto a cada quebra de linha/paragrafo novo que se adiciona um novo "P"

blockquote: resumidamente é uma tag de citação, por exemplo:

frase de alguns escritores.

U: underline que faz o papel de sublinhar frases ou palavras

mark: marca texto

sup: transcreve as letras para cima, por exemplo: dio.me r

sub: transcreve as letras para cima, por exemplo: dio.me r

hr: cria uma linha divisória na página

Atributos

Atributos das tags: atributos são propriedades que uma tag pode ter

id: identificador de elemento

style: serve para utilizar comandos "css" diretamente no elemento

Type: O atributo type é específico da tag "input"

Src: atributo que adiciona o item através de um diretorio

width: atributo de formatação

href: atributo que é adicionado a tag "a" para colocar a referencia interna ou externa

target: usado para definir se o link será aberto na mesma aba ou em outra através de 2 opções que são "_blank e _self" self é utilizada para abrir o link nela mesma ou seja na mesma página, já a o atributo blank é utilizado para abrir em outra aba do navegador.

Listas

Existem 2 listas principais as ordenadas e não ordenadas.

ol: "ordered list" ou "lista ordenada não se adicionam os itens diretamente nessa tag, precisamos das tags "filhas" que são tags que estão dentro de outras.

ul: "unordered list" ou "lista não odernada" são itens com marcadores e não são compostos por uma ordem, por exemplo: uma ordem numérica de 1, 2, 3 ...

li: "list item" ou "item da lista" são as "tags filhas" que complementam as tags "ol e ul" e pode ser feita outra lista dentro dessa tag.

Link

a: a tag de ancora é utilizada para direcionar e "ancorar" no lugar desejado.

temos alguns atributos que pode ser utilizados com ela, por exemplo: href
